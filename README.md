# My_cart
#  Smart Shopping Interface

##  Overview  
My_Cart is a modern mini e-commerce web application built using React.  
It demonstrates clean component design, effective state management, and a smooth user experience for product browsing and cart handling.

The application consumes real-time product data from an external API and provides essential shopping functionalities in an intuitive interface.

---

##  Features

- Product listing from FakeStore API  
- Search products by name  
- Filter products by category  
- Sort products by price  
- Add & remove items from cart  
- Quantity control with stock validation  
- Cart summary with total items & price  
- Persistent cart using localStorage  
- Responsive UI with modern layout
